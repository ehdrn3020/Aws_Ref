## CIDR
CIDR은 2개의 요소를 가진다.

### Base IP
EX) 10.0.0.0, 192.168.0.0 ...

### Subnet Mask
얼마나 많은 IP를 보유할 수 있는지 bits를 통해 정의한다
- /8 - 255.0.0.0
- /16 - 255.255.0.0
- /24 - 255.255.255.0
- /32 - 255.255.255.255

### CIDR 예시
192.168.0.0/24 = 192.168.0.0 ~ 192.168.255 ( 256개 )
192.168.0.0/16 = 192.168.0.0 ~ 192.255.255 ( 65,536 개 )
